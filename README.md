### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/santoshimz/pattukunte-pattucheera.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the application
   ```sh
   npm start
   ```
